# Natural Language Processing - Word Embeddings

This project covers various word embedding techniques used in Natural Language Processing (NLP).

## Tasks

### 0. Bag Of Words
Creates a bag of words embedding matrix from a list of sentences.

### 1. TF-IDF
Creates a TF-IDF embedding from a list of sentences.

### 2. Train Word2Vec
Creates, builds, and trains a gensim word2vec model.

### 3. Extract Word2Vec
Converts a gensim word2vec model to a Keras Embedding layer.

### 4. FastText
Creates, builds, and trains a gensim fastText model.

### 5. ELMo
Identifies which components are trained when building an ELMo embedding model.
